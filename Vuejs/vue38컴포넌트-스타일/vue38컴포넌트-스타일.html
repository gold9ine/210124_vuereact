Vue-CLI 을 사용하여 어플리케이션을 만들고 컴포넌트 스타일 적용 방법을 학습한다. "Vue 컴포넌트 스타일.pdf"를 참조하여 실습합니다. ################################## 1. Vue-CLI 프로젝트 생성 및 설정 작업 ################################## step 1-1. Vue-CLI 전역 설치 삭제 $ npm uninstall -g @vue/cli step 1-2. Vue-CLI 전역 설치. 한번만 수행. --force 를 사용하면 강제로 덮어쓰게 된다. $ npm install -g @vue/cli --force step 1-3. vue-cli 버전 확인 $ vue -V @vue/cli 4.5.11 step 1-4. 현재 폴더의 파일 목록 조회 $ ls vue_style_scoped_module.png vue38컴포넌트-스타일.html step 1-5. styleprj01 프로젝트 생성. package.json 생성. $ vue create styleprj01 Vue CLI v4.5.11 ? Please pick a preset: > Default ([Vue 2] babel, eslint) Default (Vue 3 Preview) ([Vue 3] babel, eslint) Manually select features step 1-6. 현재 폴더의 파일 목록 조회. 프로젝터 폴더(styleprj01) 존재 여부 확인. $ ls styleprj01 vue_style_scoped_module.png vue38컴포넌트-스타일.html step 1-7. 생성된 프로젝트 폴더(styleprj01)로 이동 $ cd styleprj01 step 1-8. 현재 폴더의 파일 목록 조회. package.json 이 보이면 정상. $ ls babel.config.js node_modules package.json package-lock.json public README.md src step 1-9. 프로젝트 실행 설정 styleprj01/package.json 수정. a. 실행시 브라우저로 열기 ==> --open b. 실행 서버 포트 설정 ==> --port 4000 "scripts": { "serve": "vue-cli-service serve --open --port 4000", "build": "vue-cli-service build", "lint": "vue-cli-service lint" }, step 1-10. 현재 폴더의 파일 목록 조회. package.json 존재하는지 확인 $ ls step 1-11. 서버 실행. package.json 이 있는 폴더에서 실행 해야 됨. $ npm run serve step 1-12. 코드 표준화 수행 $ npm run lint step 1-13. 배포할 파일을 생성. 빌드가 완료되면 dist/ 폴더에 관련 파일들이 생성됨. $ npm run build ################################## 2. 프로젝트 폴더 구조 ################################## 프로젝트 ├─ dist ├─ node_modules ├─ public │ └─ index.html │ └─ favion.ico └─ src │ └─ components │ │ └─ HelloWorld.vue # 삭제 │ │ └─ Child1.vue # 파일 생성. 스타일 기능을 테스트용 component │ │ └─ Child11.vue # 파일 생성. 스타일 기능을 테스트용 component │ │ └─ Child2.vue # 파일 생성. 스타일 기능을 테스트용 component │ │ └─ Module1.vue # 파일 생성. 스타일 기능을 테스트용 component │ ├─ App.vue │ └─ main.js ├─ babel.config.js ├─ package.json └─ README.md ################################## 3. 전역 CSS 사용시 스타일 충돌을 이해한다. ################################## step 3-1. src/components/Child1.vue 에 추가 <style> .main { border:1px solid black;
